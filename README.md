# Flutter的移动电商实战

项目和教程都在更新中............

完全模仿了百姓生活+小程序的项目，数据来自Fiddle提取，有完整的接口API文档。页面包括首页、商品列表页、商品详细页、购物车页面和会员中心。

目前项目还在开发当中，配有完整的视频和文字教程，是你Flutter进阶必备项目。
